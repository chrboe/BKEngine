Creates a single instance of Core with the given parameter.

If a instance has already been created it will be destroyed before creating a new one. If an error occurs while creating renderer and window a nullptr is returned.

Parameters
[in]widthSpecifies the width of the window to be created.
[in]heightSpecifies the height of the window to be created.
[in]windowTitleSpecifies the title of the window to be created.
Returns
A pointer to an instance of Core or nullptr if an error occurred.

◆ initEnvironment()

bool Core::initEnvironment ( )

Initializes all (directly) underlying frameworks like SDL2, SDL2_ttf**, SDL2_image and SDL2_mixer.

You may want to call this method by yourself if your code relies on the initialized third party tools but you do not need a window or a renderer yet.
Note: initEnvironment will be called automatically at Core::createInstance if it has not been called by then.

Returns
true if initialization was successful otherwise false.
