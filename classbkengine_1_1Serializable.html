void Serializable::backup ( )

Backups the current state of the object.

Through a call to Serializable::serialize the current state of the object is backupped to allow the current state to be restored later on.
Note: This overrides any backups done by Serializable::deserialize!

virtual

Uses a JSON object to set member variables.

Is used by subclasses to simplify serialization. The default implementation creates a backup of json to allow resetting the object later on.
The recommended way of using this method is following:

void SubSerialize::deserialize(const Json::Value &json)
{
Serialize::deserialize(json);
// add your own code here
}

In this way you ensure that the overridden methods will be called anyways.

void Serializable::restore ( )

Restores any backup of the object.

If any backup has been created so far (through calls to Serializable::deserialize or Serializable::backup) it will be restored with calling Serializable::deserialize. Note: This overrides does NOT override the current backup.

◆ serialize()

Json::Value Serializable::serialize ( ) const
virtual

Serializes the object to a JSON object.

Subclasses can override this method to map its members to a JSON object which allows to save it to a file and restore it later on (given that Serializable::deserialize is overridden as well!).
The default implementation returns any backup done by Serializable::deserialize or Serializable::backup (or an empty object if none of them has been called).
The recommended way of using this method is following:

Json::Value SubSerialize::serialize()
{
Json::Value json = Serialize::serialize();
// add your own code here
return json;
}

This way you ensure that all serialization done by super classes is really done!
